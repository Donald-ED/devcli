Get Started in 60 Seconds

1. Install Ollama

# Visit https://ollama.ai and install
ollama pull llama3.1

2. Install DevCLI

pip install devcli # Coming to PyPI soon!

# Or from source:
git clone https://github.com/YOUR_GITHUB_USERNAME/devcli.git
cd devcli
pip install -e .

3. Start Chatting

cd your-project/
devcli init # Scan your project
devcli # Start chatting!
